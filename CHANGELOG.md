# Changelog

All notable changes to the "MD to PDF - Blazing Fast" extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-25

### Added
- Initial release
- Convert Markdown files to PDF with a single click
- GitHub-style formatting with syntax highlighting
- Support for code blocks, tables, task lists, and ASCII diagrams
- Multiple conversion methods:
  - Editor title bar button
  - Right-click context menu
  - Status bar button
  - Keyboard shortcut (Cmd+Alt+P / Ctrl+Alt+P)
- Configurable settings:
  - Page format (A4, Letter, Legal, A3, A5)
  - Page margins
  - Auto-open after conversion
- Cross-platform browser detection (Chrome, Edge, Chromium, Brave)
